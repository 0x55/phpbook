## 撰写方法
### 文件命名
每个章节建立一个txt文件，如第11章的第3节，则建立**11.3.txt**。

### 文件头部的meta信息
每个文件头部都必须有四行：
第一行：此页面的title
第二行：前一页的章节号，没有则为0。
第三行：后一页的章节号，没有则为0。
第四行：一个#，代表meta信息结束。

### 允许的html语法
* h2、h3 代表标题。
* p、ul、li。
* `<code c>***</code> <code php>***</code>`分别用来高亮c与php的代码。

## 编译方法
编写完成后，在项目目录里执行php build.php便可。
